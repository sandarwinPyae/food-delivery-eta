# Food Delivery ETA Prediction

Setup Instructions

To run this project on your local machine, please follow these steps:

Download the Model File
Due to GitHub's file size limit (100MB), the trained model file is hosted on Google Drive. 
File Name:`multi_spectral_stack_model.pkl`
Download Link: [https://drive.google.com/file/d/1mHXp-yZx-970tf7CsVyTI_c9SJ34Tbym/view?usp=sharing]

**Note:** After downloading, please create a folder named `model` in the project directory and place the `.pkl` file inside it.

Install Dependencies

pip install -r requirements.txt
